# 🚢 Battleship Game

A browser-based Battleship game built with vanilla JavaScript.

🎮 Live demo:
https://pawelbelczyk.github.io/battleship/

## Features

- Random ship placement
- Player vs computer gameplay
- Smart AI opponent
- AI hunts ships after successful hits
- Ship sinking detection
- Game statistics
- Restart button
- Responsive design
- Unit tests with Jest

## Technologies

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Jest
- GitHub Actions
- GitHub Pages
